# User Manual for Posture Coach IoT

## 1. System Overview for End Users
The Posture Coach IoT is designed to help users maintain proper posture while seated. It offers real-time feedback and guidance to improve overall posture and wellbeing.

## 2. Quick Start Guide
1. Unbox the device and place it in your desired location.
2. Power up the device and connect it to a Wi-Fi network.
3. Download the Posture Coach application from the relevant app store.
4. Follow the setup wizard in the app to complete the configuration.

## 3. How to Connect to PostureCoach-AP WiFi Network
1. Go to your device's Wi-Fi settings.
2. Look for the network named **PostureCoach-AP**.
3. Select it and enter the password if prompted (default password: 12345678).
4. Once connected, you can proceed to the dashboard.

## 4. How to Access Web Dashboard
1. Open a web browser on your connected device.
2. Enter the dashboard URL: `http://192.168.1.1`.
3. Log in using the credentials set during the app configuration.

## 5. Dashboard Features Explanation
- **Live Posture Feedback:** View real-time data on your posture.
- **Posture History:** Access past data to track improvements over time.
- **Manual Controls:** Adjust settings manually as needed.

## 6. Understanding Posture Scores
Posture scores are calculated based on your sitting position and alignment. A higher score indicates better posture.

## 7. How to Use Manual Controls
- Navigate to the manual controls section on the dashboard.
- Adjust your settings as desired and save changes.

## 8. Break Management System Usage
The break management system prompts you to take breaks at regular intervals to help reduce fatigue and improve overall comfort.
- Set your break preferences in the dashboard settings.

## 9. Settings Configuration
You can configure various settings, including:
- Wi-Fi settings
- Notification preferences
- Posture alerts

## 10. Privacy Mode Explanation
Privacy mode disables all data transmission to ensure user privacy. Activate it in the settings to prevent data collection.

## 11. Troubleshooting Common User Issues
- **Can't connect to Wi-Fi:** Ensure the PostureCoach-AP network is selected.
- **No data in dashboard:** Confirm that the device is powered on and connected.

## 12. FAQ Section
- **Q: What should I do if the device malfunctions?**
  A: Restart the device and check the connections. 
- **Q: Can I use the device without the application?**
  A: Basic functionality is available, but full features require the app.

## 13. Maintenance Tips
- Regularly clean the device with a soft cloth.
- Update the app frequently to access new features and improvements.

## 14. System Limitations
- The device may not function optimally in environments with excessive interference or obstructions.
- Posture scores are estimates and may vary based on individual user differences.
